Descripción de los comandos y procedimientos básicos de Git.

Esta es una linea que agregué para probar los cambios.


git init Posicionarse en la carpeta en la que se desea crear el repositorio y ejecutar el comando. 
git add nombreDelArchivoAAgregar.txt agrega el archivo que está en la carpeta del repositorio o subcarpeta al resositorio


Luego de hacer cambios, podemos hace lo siguiente

git add . - agrega todos los archivos que están en la carpeta del repositorio, incluídas las carpetas

git commit -m "Comentarios de la nueva versión" Luego de agregar cambios al repositorio se agregan con este comando

git status - muestra el estado actual de la sincronización de los archivos del respositorio con respecto al servidor. 

git log nombreDelArchivo.txt -> muestra el historial de todos los cambios en el archivo indicado.
